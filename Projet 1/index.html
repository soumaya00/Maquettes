Au cours de cette formation nous allons avoir 2 matières :

L'intégration

Le HTML :

Le HTML nous sert à créer le contenu du site: elle va contenir uniquement la partie textuelle du site

Le CSS :

Le CSS va nous permettre d'ajouter la touche graphique à notre site: les couleurs, les formes, le fond, etc.

Le développement

Le JavaScript :

JavaScript est un language de programmation de script employé pour rendre une page web interactive, le comportementde la page et son apparence pourra varier en fonction de certains paramètres

Le PHP :

Le PHP est un language de programmation libre, utilisé pour générer des pages web dynamiques, qui sont donc modifiés en fonction de certaines informations(heure, nom de l'utilisateur, formulaire rempli par l'utilisateur, etc).

My SQL :

MySQL est un système de gestion de bases de données. Il permet d'enregistrer des informations remplies par l'utilisateur ( exemple: formulaire de création de compte) pour ensuite être ré-utilisé au moment approprié (ex: connexion à un site internet avec identifiant et mot de passe

400 heures de formation intensive:

  • HTML, CSS, Responsive web design, Wordpress, Bootstrap : 16 modules
  • Javascript : 12 modules
  • PHP, MySQL, POO, MVC : 25 modules
